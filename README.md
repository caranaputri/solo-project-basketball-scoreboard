# 🏀 The "I Survived This" Basketball Scoreboard

Honestly? I thought this would be a quick 20-minute exercise. Two days, five tabs of Stack Overflow, Gemini AI, a freakin' holybook,  and countless near-breakdown later... here we are. It’s a basketball scoreboard. It tracks points. It resets. 

It didn't win me a trophy, but it did made me question my life choices.

---

## 🛠️ What’s Under the Hood (and why I almost cried)

* **HTML Structure:** I spent more time naming `div` classes than I did actually watching basketball. Getting the Home and Guest sections to sit side-by-side without one of them wandering off into the void was a personal victory.
* **The CSS Struggle:** * **The "Digital" Font:** I learned that `@font-face` is a fickle friend. If the path is off by one slash, your cool "stadium" font becomes Times New Roman.
    * **Centering:** I tried using `margin: 150px` and `margin: 280px` to force things into place. My computer wanted to throw itself out the window, and I wanted to follow it.
* **The JavaScript Logic:** * I wrote functions for `+1`, `+2`, and `+3` for both teams. Yes, there is a lot of repeating code. Yes, I copy-pasted the guest functions from the home functions. Don't judge; it’s called *efficiency*.
    * The `reset()` function is the most therapeutic part of this project. Click it, and all my sins go back to zero.

---

## 🎒 Lessons Learned by a Very Tired Student

1. **Read the Errors:** If the console is red, it's usually because I forgot a closing bracket or misspelled `getElementById` for the tenth time.
2. **Save Often:** Because sometimes you change one line of CSS and the whole scoreboard disappears into a black hole.
3. **The "New Game" Button:** It’s not just for the players; it’s for me when I’ve tested the `+3` button so many times the Home team has 450 points.

---

## 🚀 How to Use (If you dare)

1. Open `index.html`.
2. Admire the fact that it actually loads.
3. Click buttons. See numbers go up. Feel the dopamine hit.
4. If it breaks, please refer back to the title of this README.
